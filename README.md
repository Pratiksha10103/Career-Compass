# 🎓 AI Career Guidance Platform

An intelligent career counseling tool built for [Hackathon Name 2026]. This platform uses **Google Gemini 2.5 Flash-Lite** to analyze student skillsets and provide personalized roadmaps.

## 🚀 Features
- **Skill Assessment:** Interactive UI to rate skills from Math to Psychology.
- **Data Visualization:** Radar charts and distribution maps of student strengths.
- **AI Counseling:** Generates 5 career paths, identifies skill gaps, and provides a 6-month learning roadmap.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Analysis:** Pandas, NumPy, Matplotlib
- **AI Engine:** Google Generative AI (Gemini API)

## 📦 Setup Instructions
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Create a `.env` file in the root directory and add your API key:
   `GOOGLE_API_KEY=your_actual_key_here`
4. Run the app:
   `streamlit run app.py`
